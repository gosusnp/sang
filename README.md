sang
====

Simple Animations for AngularJS

Requirements
------------

* jQuery
* angular.js >= 1.2.0
* angular-animate

Provides
--------

Javascript animations for angularjs.

* ``sg-fade``
* ``sg-slide-bottom``
* ``sg-slide-left``
* ``sg-slide-right``
* ``sg-slide-top``
* ``sg-uncover-horizontal``
* ``sg-uncover-vertical``
